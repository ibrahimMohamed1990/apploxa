Apploxa - whatsapp notifications to your customers

== Description ==

Apploxa enhances customer communication by automating personalized WhatsApp messages for order updates, shipment tracking, abandoned cart reminders, and more. Perfect for WooCommerce stores to streamline engagement and drive customer satisfaction.

- **Personalized WhatsApp messages**: Send automated messages to customers with dynamic variables like name, order details, and more.  
- **Automatic shipment tracking**: Share tracking links directly via WhatsApp.  
- **Abandoned cart reminders**: Notify customers about pending orders.  
- **New order notifications**: Alert staff and customers about new orders.  
- **Order status updates**: Keep customers informed on their order status.  
- **Cash on delivery confirmations**: Confirm cash on delivery orders.  
- **Welcome messages**: Greet new customers upon registration.  
- **Product review requests**: Request feedback after order fulfillment.  
- **Multiple WhatsApp numbers**: Manage communication channels easily.  
- **Message templates and scheduling**: Save time with templates and schedule messages.  

== External API ==

This plugin relies on the **Apploxa API** for sending WhatsApp messages. By using this plugin, your WordPress installation communicates with the following external service:

- **Service Name**: Apploxa API  
- **Purpose**: To send automated WhatsApp messages.  
- **Data Sent**: Customer name, phone number, order details (product names, order ID, order value), and dynamic variables for personalized communication.  
- **Trigger Conditions**: Data is sent when the plugin triggers events such as new order notifications, shipment updates, and cart reminders.  
- **Service Provider**: Apploxa  
- **Terms of Service**: [Apploxa Terms of Service](https://api.apploxa.com/terms)  
- **Privacy Policy**: [Apploxa Privacy Policy](https://api.apploxa.com/privacy)  

This information is provided to ensure transparency and compliance with WordPress plugin guidelines.

== Installation ==

1. Upload the `apploxa` folder to the `/wp-content/plugins/` directory.  
2. Activate the plugin through the 'Plugins' menu in WordPress.  
3. Configure the plugin settings on the Apploxa settings page.  

== User Creation ==

**Why does the plugin create users?**  
The plugin creates users to facilitate personalized and automated communication. For instance, it tracks customer registration to send welcome messages and log user data for subsequent notifications (e.g., order updates, cart reminders). This is essential for the plugin's functionality.
